# Vue App: Регистрация, Вход, Калькулятор

Простое веб-приложение, демонстрирующее:
- Регистрацию и вход с проверкой существующих пользователей.
- Хранение данных в `localStorage` (без бэкенда).
- Калькулятор на главной странице.
- Переключение между страницами (Главная, Вход, Регистрация).

## 🛠️ Технологии
- Vue 3 (Composition API, TypeScript)
- Vite (сборка)
- CSS (scoped)

## 🚀 Как запустить локально
1. Установите Node.js.
2. Клонируйте репозиторий.
3. Выполните `npm install`.
4. Запустите `npm run dev`.
5. Откройте http://localhost:5173.
